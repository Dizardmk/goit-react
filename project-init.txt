1) npx create-react-app .

2) npm install node-sass --save //if you want to use sass

3) npm install modern-normalize // Normalize browsers default style

4) Установить линтинг на пре-коммит
https://github.com/goitacademy/react-lint-config
- npm install husky --save-dev
- npx husky install
- npx husky add .husky/pre-commit "npx lint-staged"

Создать файлы в корне проекта:

.huskyrc
{
  "hooks": {
    "pre-commit": "lint-staged"
  }
}

.lintstagedrc
{
  "src/**/*.{json,css,scss,md}": ["prettier --write"],
  "src/**/*.{js,jsx,ts,tsx}": ["prettier --write", "eslint --fix"]
}

.prettierrc
{
  "printWidth": 80,
  "tabWidth": 2,
  "useTabs": false,
  "semi": true,
  "singleQuote": true,
  "trailingComma": "all",
  "bracketSpacing": true,
  "jsxBracketSameLine": false,
  "arrowParens": "avoid",
  "proseWrap": "always"
}

5) Для комфортной работы, после установки плагинов, нужно добавить несколько настроек редактора для автосохранения и форматирования файлов.
{
  "files.autoSave": "onFocusChange",
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  }
}

6) Деплой приложения ghpages
https://create-react-app.dev/docs/deployment#github-pages
https://drive.google.com/file/d/1EOewQyS7V9SHsUbbycwgTNqB59jwhFnG/view

- добавить <"homepage": "https://myusername.github.io/my-app",> в package.json
- npm install --save gh-pages
- добавить скрипты в pachage.json:
"predeploy": "npm run build",
"deploy": "gh-pages -d build",
